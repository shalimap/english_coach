English Coach App

A multilingual mobile application built using Flutter to help users improve English through structured modules, audio content, and interactive features.

🚀 Features
🌍 Multilingual support (English & Malayalam)
🔐 User authentication & profile management
🔔 Push notifications using Firebase
🎧 Audio-based learning modules
💬 Audio chat & interactive features
📡 REST API integration
⚡ Smooth performance with optimized UI
🏗️ Architecture
Clean folder structure
Separation of concerns (API, Services, UI, Models)
Reusable widgets for scalability
🛠️ Tech Stack
Flutter & Dart
Firebase (Messaging, Notifications)
REST APIs
Provider (State Management)
